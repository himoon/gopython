## 1. 사전 준비 (가상환경 설정 필수)
패키지를 설치하기 전에 **반드시 파이썬 가상환경을 먼저 설정해야 합니다.** 가상환경 없이 패키지를 설치하면 시스템 전역에 설치되어 프로젝트마다 패키지가 충돌하거나 관리가 어려워질 수 있습니다.

- 아직 가상환경을 설정하지 않았다면 아래 문서를 먼저 참고하여 설정을 완료해 주세요.
  - [VS Code 파이썬 가상환경 설정](./venv-setup.md)

> ⚠️ **윈도우 사용자 주의:** 윈도우 환경에서는 터미널 기본 프로필을 **파워셸(PowerShell)이 아니라 명령 프롬프트(Command Prompt)로 변경**해야 가상환경이 제대로 활성화됩니다. 자세한 방법은 [가상환경 설정 문서의 "터미널 기본 프로필 변경" 단계](./venv-setup.md#2-터미널-기본-프로필-변경-윈도우-사용자)를 참고하세요.

## 2. 터미널 열기
VS Code 상단 메뉴에서 터미널을 엽니다.

- 상단 메뉴에서 `터미널(Terminal)` 클릭 후 `새 터미널(New Terminal)`을 선택합니다.
- 키보드 단축키 `` Ctrl + Shift + ` ``를 눌러도 새 터미널을 열 수 있습니다.

## 3. pip 명령어로 패키지 설치하기
터미널이 열리면 설치할 패키지를 `pip install` 명령어와 함께 입력합니다.

- 예를 들어 `pandas` 패키지를 설치하려면 아래 그림처럼 다음과 같이 입력하고 `Enter` 키를 누릅니다.

  ```bash
  pip install pandas
  ```

  <img src="https://raw.githubusercontent.com/himoon/gopython/refs/heads/main/images/pip-install-01.png" alt="터미널에서 pip install pandas 입력" width="700"/>

> 💡 **가상환경이 제대로 설정되었는지 확인하는 방법:** 위 그림처럼 터미널 프롬프트 앞에 괄호로 `(.venv)` 또는 `(폴더명)`이 표시되면 가상환경이 정상적으로 활성화된 것입니다. 이 표시가 보이지 않는다면 [가상환경 설정 문서](./venv-setup.md)를 다시 확인해 주세요.
